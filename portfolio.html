Tecnologías clave

Mi stack principal

Construyo proyectos con JavaScript, pensado para rendimiento, escalabilidad y despliegue rápido y economico.

01

React + Vite

Interfaces reactivas con una experiencia de usuario fluida. Vite acelera el desarrollo y las cargas, manteniendo la app ligera.

02

Node.js + Serverless

APIs rápidas y mantenimiento sencillo con JavaScript en el backend. Deploy serverless en Vercel para escalabilidad instantánea.

03

Firebase

Autenticación, base de datos en tiempo real, almacenamiento y servicios de hosting. Ideal para proyectos ecommerce modernos que necesitan datos confiables y entrega rápida.

04

Vercel

Despliegue continuo sin complicaciones. Desde el código hasta el sitio en línea, la infraestructura es rápida, estable y preparada para cambios constantes.

05

JavaScript Full Stack

Todo el flujo del proyecto se resuelve con JavaScript: frontend, backend y servicios integrados. Comunicación directa entre capas y menos fricción de implementación.
